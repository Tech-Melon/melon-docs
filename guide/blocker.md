# 🚫 内容拦截器

内容拦截器让你自定义「不想在信息流里看到」的内容。规则保存在 **当前浏览器本地**，换设备需要重新配置。

打开方式： [超级工具箱](/guide/toolbox) → **🚫 内容拦截**。

![内容拦截面板](/screenshots/2026-07/05-toolbox-blocker.webp)

---

## 1. 规则维度

添加规则时，至少填写 **一个** 条件（全空规则会被拒绝，防止误杀全部内容）：

| 维度 | 说明 |
|------|------|
| **平台** | 全平台 / 微博 / 公众号 / 抖音 / Ins / TK / YouTube / 快讯 等 |
| **账号名** | 支持精确匹配或模糊包含（按钮切换） |
| **关键词** | 在正文、标题、作者等文本中模糊匹配 |
| **AI 标签** | 匹配 AI 话题标签文本 |

规则可启用 / 停用 / 编辑 / 删除，顶部支持搜索已有规则。

---

## 2. 生效范围（必读）

| 场景 | 是否拦截 |
|------|----------|
| 实时推送的主信息流卡片 | ✅ 拦截（不渲染） |
| 快讯侧栏条目 | ✅ 拦截 |
| 历史分页加载 | ✅ 拦截 |
| **聚合搜索结果** | ❌ **不拦截**（主动溯源优先） |

若你需要「被拦截账号」的某条旧消息，请用 [聚合搜索](/guide/search) 按关键词或账号找回。

---

## 3. 推荐用法

- **烦某博主**：平台 + 账号（精确）  
- **烦某类广告话术**：关键词  
- **烦某类 AI 标签**：AI 标签字段  
- **组合**：账号 + 关键词（同时满足才拦，更精准）

::: tip 小技巧
规则越具体越安全。优先用「账号 + 关键词」而不是单个极短关键词，避免误伤。
:::

---

## 4. 与 AI 过滤的区别

| | AI 过滤 | 内容拦截 |
|--|---------|----------|
| 谁定义规则 | 系统模型打分 | 你自己写规则 |
| 开关 | 顶部 **AI: 开/关** | 工具箱内规则启用 |
| 目的 | 批量降噪生活帖 | 精准屏蔽特定源/词 |

可以同时开启：AI 负责「整体变干净」，拦截负责「点名拉黑」。
