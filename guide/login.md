# 🔐 第一步：登录系统

## 1. 获取访问密钥

本系统采用 **订阅制 (Subscription)**。
如果您还没有密钥，请先查阅 [💰 订阅说明](/guide/intro#如何订阅) 获取密钥。

::: danger ⚠️ 安全警告

- **密钥即账号**：您的密钥绑定了您的订阅有效期与权限，请妥善保管。
  :::

## 2. 登录步骤

1.  **访问地址**：打开浏览器访问 [https://weibo.tech-melon.top](https://weibo.tech-melon.top)。
2.  **输入密钥**：在输入框中粘贴您的 登录密钥。
    - _小技巧_：点击输入框右侧的 **👁️ (眼睛图标)** 可以显示明文，方便核对是否多复制了空格。
3.  **进入系统**：点击 **“立即进入”** 按钮。

## 🆘 常见问题 (FAQ)

::: warning Q1：为什么我突然被踢下线了？  
**原因：系统触发了单设备保护机制。**  
为了保障情报的时效性与账号安全，系统限制 **单个密钥同一时间只能在一台设备在线**。

- 如果您在 **手机** 上登录，原本在线的 **电脑端** 会自动断开连接。
- **例外情况**：同一个浏览器内打开多个标签页（多开窗口）**不会**互踢。

:::

::: tip Q2：为什么新消息没有声音提醒？  
**原因：浏览器的自动播放策略 (Autoplay Policy)。**  
现代浏览器（Chrome/Safari/Edge）默认禁止网页在用户没有交互时自动播放声音。

**✅ 解决方法：**  
登录系统后，请务必 **点击页面任意空白处** 或 点击右上角设置里的 **“▶ 测试”** 按钮。

- 当您看到顶部状态栏显示 <span style="color:#52c41a">**“声音已就绪”**</span> 或听到提示音后，后续的新消息就会自动报警了。

:::

::: details Q3：提示“无效的密钥”怎么办？

1.  请检查密钥前后是否复制了**多余的空格**。
2.  请确认您的订阅 **是否已过期**。
3.  如果确认无误，请联系管理员核实后台状态。

:::

::: danger 🆘 紧急救援：连接不上 / 一直转圈？
如果您登录后发现页面顶部一直显示 **"🔄 正在连接服务器..."**，或者没有任何数据加载：

1.  **强制刷新**：
    - **Windows**: 按下 `Ctrl` + `F5`
    - **Mac**: 按下 `Cmd` + `Shift` + `R`

2.  **清理缓存重登 (最有效)**：
    - 点击页面右上角的 **"?" (帮助)** 按钮，打开系统急救面板。
    - 点击底部的 **“⚠️ 无法连接？点此清理缓存...”** 按钮。

    ![系统急救面板示意图](/clean_cache_guide.png?v=1.1)
    <p align="center" style="color: #999; font-size: 0.9em;">图示：通过右上角 "?" 按钮打开内置修复工具</p>

3.  **检查网络代理**：\* 本系统使用 WebSocket 协议。某些 VPN 或代理软件可能会拦截该连接，请尝试 **关闭代理** 或 **切换节点** 后重试。

:::
