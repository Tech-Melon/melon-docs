# 🔐 第一步：登录系统

## 1. 获取访问密钥

本系统采用 **订阅制**。若还没有密钥，请先阅读 [产品简介 · 如何订阅](/guide/intro#如何订阅)。

::: danger ⚠️ 安全警告
**密钥即账号**：绑定订阅有效期与权限，请勿公开分享。
:::

## 2. 登录步骤

1. 浏览器打开 [https://weibo.tech-melon.top](https://weibo.tech-melon.top)  
2. 在输入框粘贴访问密钥  
3. 可用 **👁️** 查看明文，避免首尾空格  
4. 点击 **立即登录进入**

![登录界面](/screenshots/2026-07/00-login.webp)

## 🆘 常见问题

::: warning 突然被踢下线？
系统限制 **同一密钥同一时间主要设备单端在线**（同浏览器多标签一般不互踢）。  
手机登录可能挤掉电脑端，反之亦然。
:::

::: tip 没有提示音？
浏览器禁止自动播放。登录后点击页面空白处，或在 ⚙️ 里点 **▶ 测试**。详见 [声音与信源开关](/guide/audio)。
:::
