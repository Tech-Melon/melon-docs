# 🎴 读懂监控卡片

信息流里每一张卡片都用 **颜色、图标、摘要** 帮你快速判断优先级。

## 1. 主站平台：边框颜色

| 颜色 | 平台 | 含义 |
|------|------|------|
| 🟠 橙色 | 微博 | 实时动态主体 |
| 🟢 绿色 | 微信公众号 | 长文 / 多图文组 |
| 🔴 抖音粉 | 抖音 | 短视频情报 |
| 🟣 紫粉 | Instagram | 海外图文 |
| ⚫ 黑灰 | TikTok | 海外短视频 |
| 🔴 YouTube 红 | YouTube | 视频 / 频道帖子 |
| 🔴 左边框高亮红 | 任意平台 | **高分预警**（AI 分数很高） |

## 2. 快讯卡片

快讯使用信源 **彩色角标**（律动绿、币安金、X狙 橙等），与社媒卡片区分。

| 类型 | 特征 |
|------|------|
| 律动快讯 | 「律动快讯」角标 + 标题/正文 |
| 律动文章 | 「📄 文章」+ 摘要 |
| Flap / Four | 新币、审计、玩法结构 |
| 币安 | 金色角标，可按公告分类 |
| **X狙** | 作者 handle、动作（发推/转发/回复）、正文、CA、可选译文 |

详见 [快讯与情报源](/guide/newsflash)。

![X狙 卡片](/screenshots/2026-07/11-xbot-snipe-cards.png)

## 3. AI 智能辅助

### 🏷️ 话题图标

标题旁的 🏛️ / ⚛️ / 📉 等表示 AI 归类方向（宏大叙事、硬核科技、市场异动…）。

### ⚡ 分数

分数越高越值得优先看；高分卡片常带红色强调边框。

### 🧠 摘要框

浅色分析区给出一句话总结与推荐理由，适合快速扫读。

### 🤖 中文自动翻译

海外与 X狙 外文内容会在卡片内异步插入译文，无需刷新。

## 4. 特殊标识

- **央视等官媒印章**：高可信来源标识  
- **改名提醒**：监控对象改昵称时的系统卡片  
- **NEW**：刚推送的新鲜内容
