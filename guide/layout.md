## 1. 系统内的双模

| 模式 | 说明 |
|------|------|
| **Mode 1 宽屏** | 窗口足够宽时，右侧独立快讯栏 |
| **Mode 2 合流** | 窄屏、折叠快讯，或开启「大屏快讯入流」时，快讯进入中间流 |

设置：右上角 **⚙️** → **大屏快讯入流**。详见 [快讯与情报源](/guide/newsflash)。

## 2. 浏览器分屏（Chrome 推荐）

### 第一步：独立窗口

按住技术瓜标签页向下拖，变成独立窗口。

### 第二步：贴靠屏幕

#### Windows

1. 主工作窗口：`Win` + `←`  
2. 技术瓜窗口：`Win` + `→`  
3. 拖中间分割线，技术瓜约占 1/4～1/3 宽  

#### Mac

1. 悬停窗口绿色按钮  
2. 拼贴到左侧 / 右侧  
3. 拖分割条调宽  

---

## 3. 显示张数

⚙️ → **显示设置**：可调整「分类浏览」每类张数与聚合总张数，避免列表无限膨胀卡顿。

---

## 4. 建议工作流

1. 左侧：交易 / K 线 / 推特  
2. 右侧或侧栏：技术瓜（AI 开 + 关键源提示音开）  
3. 突发时用 [聚合搜索](/guide/search) 溯源  
4. 噪音源用 [内容拦截](/guide/blocker) 处理
