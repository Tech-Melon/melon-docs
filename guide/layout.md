# 📺 打造极致盯盘桌面 (分屏设置)

**技术瓜实时监控 Pro** 的 UI 采用了全响应式设计（Responsive Design）。这意味着，它不仅可以全屏展示，还可以像手机一样**变窄**，完美地挂在屏幕侧边。

通过“分屏展示”，您可以一边看 K 线、推特或其它工具，一边通过技术瓜接收实时舆情，实现 **信息流与交易流的同屏共振**。

## 🛠️ 如何设置 (Google Chrome)

我们推荐使用谷歌浏览器的 **独立窗口** 功能来实现这一布局。

### 第一步：将技术瓜“独立”出来

1.  在浏览器中打开 **技术瓜监控页面**。
2.  用鼠标按住顶部的**标签页 (Tab)**，将其向下拉拽，直到它脱离当前浏览器窗口，变成一个独立的浮窗。

### 第二步：调整布局 (Windows/Mac)

#### 🪟 Windows 用户

1.  点击选中您的**主工作窗口**（如 GMGN、推特或 TradingView），按下键盘 `Win` + `←` (左方向键)，将其贴靠在屏幕左侧。
2.  点击选中**技术瓜独立窗口**，按下键盘 `Win` + `→` (右方向键)，将其贴靠在屏幕右侧。
3.  鼠标拖动两个窗口中间的分割线，将技术瓜窗口调整至适合阅读的宽度（建议占屏幕 1/4）。

#### 🍎 Mac 用户

1.  将鼠标悬停在浏览器左上角的 **绿色全屏按钮** 上。
2.  在弹出的菜单中选择 **“将窗口拼贴到屏幕左侧”**。
3.  在屏幕右侧选择技术瓜窗口。
4.  拖动中间的黑色分割条调整宽度。

## ✨ 智能双模流转交互 (四象限自适应)

新版引入了业内顶尖的**“快讯聚合模块交互”**，彻底解决了侧边栏信息打扰与阅读面积的冲突。所有布局切换**完全免刷新**，即时热更新：

### 1. 小屏/侧边栏模式 (智能接管)
当您将浏览器宽度缩窄（例如贴靠在屏幕一侧）时：
- 右侧庞大的专属浮窗会自动消失。
- 所有的快讯提醒将**瞬间流入中间的主信息流**，确保在狭小空间内您的视野依旧干净且不漏关键情报。

### 2. 大屏/全屏模式 (自由裁量)
在宽屏状态下，您拥有绝对的控制权：
- **沉浸浮窗模式 (默认)**：快讯在右侧拥有专属大面板，新消息到来时会弹出不影响主视线的“吐司(Toast)”弹窗。
- **强制入流模式**：如果您觉得面板太占地方，点击右上角的 `[✖]` 或在设置中开启“大屏快讯入流”，庞大的浮窗会立即缩为一个灵动的 `[⚡️]` 按钮。此时，快讯会自动降级混入中间的主信息流中，吐司弹窗也被智能屏蔽，还您最大的沉浸阅读空间！

::: tip 💡 极客技巧
如果您拥有多个显示器，我们强烈建议将技术瓜监控页面 **单独放置在一个竖屏显示器** 上，开启**大屏强制入流**，配合 **[全量原始模式](/guide/filter)**，即可获得类似彭博终端 (Bloomberg Terminal) 的专业新闻流体验。
:::
