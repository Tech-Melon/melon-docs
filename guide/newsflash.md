# ⚡ 快讯与情报源

快讯模块是系统的「高频情报侧栏 / 入流卡片」集合，覆盖律动、币安、链上与 **X狙** 等信源。宽屏下默认出现在右侧面板；窄屏或开启「大屏快讯入流」后会进入中间主信息流。

![快讯面板（宽屏）](/screenshots/2026-07/07-newsflash-panel.webp)

## 1. 双模布局

| 模式 | 条件 | 表现 |
|------|------|------|
| **右侧独立栏** | 窗口宽度 ≥ 约 1360px，且未强制「大屏快讯入流」 | 右侧专属滚动列表，中间主信息流以社媒为主 |
| **主信息流入流** | 窄屏，或设置中开启「大屏快讯入流」，或折叠右侧栏 | 快讯以卡片形式出现在中间聚合流 |

设置路径：右上角 **⚙️** → **大屏快讯入流**。

---

## 2. 信源一览

### 主站平台（中间信息流 Tab）

微博 · 公众号 · 抖音 · 海外（Ins / TikTok / **YouTube**）

### 快讯矩阵（侧栏 + 可入流）

| 信源 | 你能看到什么 |
|------|----------------|
| **律动快讯 / 文章** | BlockBeats 快讯标题正文；文章摘要与作者 |
| **42Space / 上链** | 预测市场与链上同步状态 |
| **Flap / Four** | 新币、审计、玩法提醒 |
| **币安专区** | 公告、配置、TradFi、Zones 等（侧栏可切「币安专区」+ 分类） |
| **SEC IPO** | 招股相关动态 |
| **狗妈** | 专属监控源 |
| **🎯 X狙** | Twitter/X 推文结构化卡片：作者、动作（发推/转发/回复）、CA、媒体；支持 **中文自动翻译** |

侧栏顶部有 **全部 / 律动快讯 / 币安专区** 等 Tab，可快速过滤。

---

## 3. X狙 是什么？

X狙（`xbot_snipe`）用于监控指定推特账号的发推、转发、回复等行为，前端用结构化卡片展示，而不是一整坨纯文本。

![X狙 结构化卡片](/screenshots/2026-07/11-xbot-snipe-cards.webp)

<p align="center"><i>图：X狙 搜索结果 · 作者 / 动作 / 正文 / 译文</i></p>

**使用提示：**

1. 在 **⚙️ 设置** 里可为「🎯X狙」单独开关 **提示音** 与 **数据接收**。
2. 外文推文会异步翻译，完成后卡片出现「中文自动翻译」块，无需刷新。
3. 想翻历史：用 [聚合搜索](/guide/search)，勾选 **⚡ 快讯**，关键词可用推文原文、中文译文或 `@handle`。

## 4. 与 AI 过滤的关系

快讯默认视为高价值情报，**不会被 AI 垃圾过滤折叠**。若你在信息流里「看不到某类快讯」，请优先检查：

1. 设置里该信源的 **数据接收** 是否关闭  
2. [内容拦截器](/guide/blocker) 是否命中账号/关键词  
3. 是否处于某个平台 Tab / 赛道筛选（主动 [搜索](/guide/search) 不受拦截与 Tab 限制）
