# 🔍 聚合搜索

聚合搜索用于 **主动溯源**：在指定时间范围与平台内，按关键词或账号找回历史情报。

打开方式： [超级工具箱](/guide/toolbox) → **🔍 聚合搜索**。

![搜索面板](/screenshots/2026-07/02-toolbox-search.webp)

---

## 1. 搜索过滤器（唯一约束）

请先点搜索框旁的 **⚙️** 展开高级参数：

| 参数 | 说明 |
|------|------|
| **关键词** | 匹配正文 / 标题 / 译文等（见下表） |
| **定向账号** | 按作者/账号模糊匹配 |
| **时间范围** | 24 小时 ~ 完整生命周期 |
| **AI 打分** | 最低分数门槛（社媒 AI 帖）；快讯按默认高分处理，不会被默认门槛整类滤掉 |
| **监控平台** | 微博 / 公众号 / 抖音 / Ins / TK / YT / ⚡快讯 |

偏好会自动保存在浏览器本地，下次打开会恢复。

---

## 2. 各平台实际匹配什么

| 平台 | 关键词大致匹配 |
|------|----------------|
| 微博 / 抖音等 | 正文、描述 |
| 公众号 | 标题、摘要 |
| YouTube | 标题 / 帖子正文 |
| **⚡ 快讯（含 X狙）** | 标题、正文、**中文译文**、作者、推文 `diff_payload` 正文与 handle |

搜 X狙 时建议：

- 勾选 **⚡ 快讯**
- 关键词用推文里的词、中文译文，或 `@handle`
- 也可用「定向账号」填 handle / 昵称片段

## 3. 重要：搜索 vs 内容拦截 vs AI 过滤

| 规则 | 实时信息流 | 主动搜索结果 |
|------|------------|--------------|
| AI 垃圾过滤 / 平台 Tab / 赛道 | ✅ 生效 | ❌ **不限制展示** |
| 内容拦截器 | ✅ 生效 | ❌ **不屏蔽** |
| 搜索面板上的关键词/平台/天数/AI 分 | — | ✅ 唯一过滤条件 |

设计原因：拦截是「被动降噪」；搜索是「我主动要找」。两者目标不同。

---

## 4. 退出搜索

结果页顶部会出现搜索横幅。点击 **✖ 退出搜索 (恢复实时)**，即可回到实时信息流。

---

## 5. 常见情况

**Q: 明明信息流里见过，搜索却没有？**  
A: 检查是否勾选对应平台、时间范围是否够长；X狙 请勾选快讯，并用正文/译文/handle 试一次。

**Q: 条数很多但列表偏少？**  
A: 后端单次有结果上限；可收窄关键词或时间再搜。
